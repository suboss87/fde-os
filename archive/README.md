# Archive

Pre-v3 material, kept for reference and recoverable via git history.

- `skills/` — the original 16 standalone skills. In v3 their content was
  consolidated into **one** skill (`skills/fde/`) with per-phase reference
  files (`skills/fde/references/`), restructured so each phase does work and
  writes an artifact instead of giving advice.
- `templates-phase/` — `.fde/` templates for files that phases now create on
  demand (their formats live inside the phase references).

Nothing here is installed by `bin/install.js`.

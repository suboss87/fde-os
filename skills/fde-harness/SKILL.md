# FDE Harness - Skill Factory

## Purpose
The best FDEs build platforms, not just solutions. This skill enables self-improvement.

## When Activated
- You've done something twice and see a pattern.
- User says "create a skill for X" or "automate this pattern".

## Harness Protocol
1. **Identify Pattern**: what repeated? What would a skill need to do?
2. **Encode as SKILL.md**: follow FDEOS format: YAML, Purpose, When Activated, Protocol, Iron Rules.
3. **Test**: run the new skill on a known problem.
4. **Register**: add to project's CLAUDE.md.
5. **Propose Contribution**: if general-purpose, open a PR to FDEOS.

## Iron Rule
**If you do it twice, encode it. If you encode it, test it. If it passes, share it.**

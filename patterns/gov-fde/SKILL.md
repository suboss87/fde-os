# Government FDE Pattern

## Added Rules
- **FedRAMP**: Services must comply with moderate/high baseline controls.
- **Authority to Operate (ATO)**: Any system change that crosses the security boundary requires re-authorisation. Flag changes explicitly.
- **Security Boundaries**: Map inter-system connections; any new connection must be documented.
- **Data Sovereignty**: Ensure data remains within authorised jurisdictions.
